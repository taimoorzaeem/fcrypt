# File-Encryption-and-Decryption-Tool

A command line tool to encrypt and decrypt a file using a symmetric key.

**To build:**

```
$ make
```

**To install:**

```
$ sudo make install
```

**Using the tool:**

To encrypt a file:

```
$ fcrypt -e <filename> <key-filename>
```

To decrypt the same file:

```
$ fcrypt -d <filename> <key-filename>
```
